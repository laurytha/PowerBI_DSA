A Análise Comercial tem como objetivo compreender e otimizar o desempenho de uma organização na área de vendas, visando atingir as metas de negócios, impulsionar a lucratividade e melhorar a eficiência operacional. A área comercial é responsável pela relação com os clientes, desde a prospecção até o pós-venda. Por esse motivo, é fundamental estabelecer estratégias de vendas, realizar negociações, fechar contratos e acompanhar o desempenho das vendas.

Neste projeto o objetivo foi para desenvolver habilidades para a análise comercial oferecendo uma visão da performance de vendas a partir da transformação de dados brutos em informações valiosas para impulsionar decisões estratégicas. 

Para realizar as análises, utilizei o dataset (com dados fictícios) obtidos no curso com a DSA Data Science Academy, com os principais requisitos:

Total de Venda por Segmento
Total de Venda por Fabricante
Total de Venda por Categoria
Total de Venda por Categoria e Ponto de Venda(PDV)
Análise dos principais influenciadores de Venda
Análise Geoespacial com performance do Vendedor

Vale ressaltar que a diagramação foi elaborada especialmente para esse dashboard.
